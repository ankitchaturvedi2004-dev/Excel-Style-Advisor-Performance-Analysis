# 📊 Style Advisor Performance Analysis (Excel Project)

## 📌 Objective
Analyze the performance of retail style advisors across different zones to identify trends, high performers, and improvement areas.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Conditional Formatting
- Nested IF Formulas
- Charts (Bar, Pie, Line, Stacked)

## 📂 Dataset
Includes evaluation data with:
- Date
- Zone / Region
- Store
- Style Advisor
- Evaluation Scores
- Performance Grades

## 📈 Key Analysis Performed
- Performance grading using IF formulas
- Zone-wise comparison using pivot tables
- Trend analysis over time
- High vs Low performer segmentation

## 💡 Key Insights
- Majority of advisors fall under average performance
- Some zones consistently outperform others
- Performance fluctuates over time
- Gap exists between high and low performers

## 📊 Visualizations
- Bar Chart: Average scores by zone
- Pie Chart: High performer distribution
- Line Chart: Performance trends
- Stacked Bar: High vs Low performers

## 🚀 Business Recommendations
- Provide targeted training programs
- Replicate best practices from top zones
- Improve low-performing criteria
- Continuous monitoring of performance
